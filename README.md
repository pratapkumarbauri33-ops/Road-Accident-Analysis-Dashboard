## Road Accident Data Analysis Dashboard 🚦📊

This project focuses on analyzing road accident data using Excel dashboards and data visualization techniques. The dataset contains detailed information about accident severity, road conditions, weather conditions, vehicle types, casualties, and other important factors.

## 🔍 Project Objectives
- Analyze accident trends and patterns
- Identify high-risk road and weather conditions
- Compare accident severity levels
- Study vehicle involvement in accidents
- Visualize monthly and yearly accident statistics
- Create an interactive dashboard for better insights
- 
## 📁 Project Contents
 - Raw Accident Dataset
 - KPI Analysis
 - Interactive Dashboard
 - Monthly Trend Analysis
 - Road Type Analysis
 - TreeMap Visualization

## 📊 Tools Used
 - Microsoft Excel
 - Pivot Tables
 - Charts & Graphs
 - Data Cleaning Techniques
 - Dashboard Design

## 📈 Key Insights
  - Accident severity distribution
  - Casualty analysis by vehicle type
  - Impact of weather and road conditions
  - Monthly accident trends
  - High accident-prone road categories
   
## 🎯 Outcome

This project helps in understanding accident patterns and supports data-driven decision making for improving road safety and traffic management.
